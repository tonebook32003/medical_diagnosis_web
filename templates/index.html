Hỗ trợ chẩn đoán & gợi ý điều trị

Ứng dụng minh họa khai thác dữ liệu: từ bộ dữ liệu triệu chứng và nhãn y khoa, hệ thống khai thác tập phổ biến và luật kết hợp để gợi ý chẩn đoán, mức độ nghiêm trọng và phương án điều trị tương ứng.

Ngôn ngữ & thư viện

Python 3 Flask (web) HTML / CSS JavaScript (fetch API) Pandas mlxtend (FP-Growth, luật kết hợp)
Luồng xử lý (workflow)
  1. Nạp file CSV (cùng cấu trúc với dataset của bạn).
  2. Mỗi bản ghi được chuyển thành một “giao dịch”: triệu chứng, chẩn đoán, mức độ, phương án điều trị (có tiền tố để phân biệt loại mục).
  3. FP-Growth tìm tập mục phổ biến; sinh luật kết hợp với ngưỡng support và confidence bạn chọn.
  4. Bạn chọn triệu chứng hiện có → hệ thống khớp vế trái luật và xếp hạng gợi ý chẩn đoán; kết hợp thêm gợi ý mức độ / điều trị (khi luật khớp).
